Arestas - edges¶

grafos.edges.getAllArcs(self)[código fonte]¶

Obtém todos os arcos do grafo.

Percorre a matriz de valores do grafo, inserindo todas os arcos do grafo formatadas no padrão [node1, node2, weight] em uma lista. Lembrando que uma aresta == 2 arcos de mesmo valor em direção opostas

Args:
None
Returns:
edges (list): lista contendo todos os arcos do grafo, formatados e não repetidas.
Raises:
None
